# 📊 Student Performance Dashboard (RStudio)

## 📖 Project Overview

This project recreates a Student Performance Dashboard using RStudio as part of Data Modelling & Visualization .

The objective is to perform Exploratory Data Analysis (EDA) and generate meaningful insights using statistical visualization techniques.

---

## 📂 Dataset

**Dataset Source:**  
[Students Performance in Exams - Kaggle](https://www.kaggle.com/datasets/spscientist/students-performance-in-exams)

The dataset contains student scores in:
- Math
- Reading
- Writing

Along with demographic and background information.

---

## 🛠 Tools & Technologies Used

- R
- RStudio
- R Markdown
- ggplot2
- dplyr
- corrplot

---

## 📊 Analysis Performed

- Data Cleaning
- Feature Engineering (Total & Average Score)
- Gender-wise Performance Analysis
- Test Preparation Impact Analysis
- Parental Education Impact
- Correlation Analysis
- Score Distribution Visualization
- Grade Classification

---

## 📈 Key Insights

- Students who completed test preparation scored higher.
- Strong positive correlation exists between all three subjects.
- Parental education level shows positive association with performance.
- Female students perform better in reading and writing subjects.
